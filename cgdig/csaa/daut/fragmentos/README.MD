# Criador de Fragmentos

App para auxiliar a seleção de fragmentos para a criação de cenários Workflow baseado nos conujuntos de opções selecionadas.  
Inicialmente desenvolvida para o serviço TPU, poderá ser expandida para outros futuramente.  
Desenvolvidas as versões texto (linha de comando) e GUI.
